postSpeakerRole

Set the role of a speaker

URL

http://localhost/api/v1/audio/speakers/{speakerId}/role

HTTP Method

POST

Response Type

Speaker

Parameters

  • Path: speakerId String

    Id of the speaker which is assigned a new role

  • Body: SpeakerRole SpeakerRole(SpeakerRole)

    roleId corresponding to the id returned by /audio/roles
