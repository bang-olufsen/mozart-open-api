getPqItems

Get playqueue items

URL

http://localhost/api/v1/playback/queue

HTTP Method

GET

Response Type

PlayQueueItemArray

Parameters

  • Query: after_id UUID

    get queue after specified queue id

  • Query: count Integer

    get count number of items. Default value is 20. Minimum value is 1 and maximum is 1000

  • Query: details Boolean

    set to true if all metadata is wanted. When false or not specified only id, type and uri/trackId is returned
