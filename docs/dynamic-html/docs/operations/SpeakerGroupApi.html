This is the API

deleteSpeakergroup

Delete the speaker group with given id. If the active speaker group is deleted, the first speaker group in the list will automatically be made active.

URL

http://localhost/api/v1/sound/speakergroup/{id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: id UUID

    ID of speaker group to delete

getSpeakergroup

Get the speaker group with the given id

URL

http://localhost/api/v1/sound/speakergroup/{id}

HTTP Method

GET

Response Type

SpeakerGroup

Parameters

  • Path: id UUID

    ID of the speaker group to get

getSpeakergroupActive

Get the active speaker group

URL

http://localhost/api/v1/sound/speakergroup/active

HTTP Method

GET

Response Type

SpeakerGroupOverview

Parameters

getSpeakergroups

Get list of configured speaker groups

URL

http://localhost/api/v1/sound/speakergroups

HTTP Method

GET

Response Type

SpeakerGroupList

Parameters

modifySpeakergroup

Modify speaker group with given id

URL

http://localhost/api/v1/sound/speakergroup/{id}

HTTP Method

PUT

Response Type

Parameters

  • Path: id UUID

    ID of the speaker group to modify

  • Body: SpeakerGroup SpeakerGroup(SpeakerGroup)

    Settings for given speaker group. Multiple settings can be applied at once.

postSpeakergroup

Add a new speaker group

URL

http://localhost/api/v1/sound/speakergroup

HTTP Method

POST

Response Type

SpeakerGroup

Parameters

  • Body: SpeakerGroup SpeakerGroup(SpeakerGroup)

    Friendly name and optionally speaker group data for the new speaker group.

postSpeakergroupActivate

Set active speaker group

URL

http://localhost/api/v1/sound/speakergroup/{id}/activate

HTTP Method

POST

Response Type

Parameters

  • Path: id UUID

    ID of the speaker group to activate
