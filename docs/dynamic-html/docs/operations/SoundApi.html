getListeningModeSet

Get available listening modes. By default you will only get listening modes that match the current role (standalone or multichannel), but you can use the `?all=true` query parameter to get listening modes for all roles.

URL

http://localhost/api/v1/sound/listening-modes

HTTP Method

GET

Response Type

List

Parameters

  • Query: all Boolean

    Get listening modes for all speaker roles

getSoundFeatures

Get available sound features. By default you will only get sound features that match the current role (standalone or multichannel), but you can use the `?all=true` query parameter to get sound features for all roles.

URL

http://localhost/api/v1/sound/features

HTTP Method

GET

Response Type

Map

Parameters

  • Query: all Boolean

    Get sound features for all speaker roles
