This is the API

addAlarm

Add a configured alarm to the product

URL

http://localhost/api/v1/alarm

HTTP Method

POST

Response Type

AlarmInstance

Parameters

  • Body: AlarmInstance AlarmInstance(AlarmInstance)

    New alarm instance

addTimer

Add a new timer to the product

URL

http://localhost/api/v1/timer

HTTP Method

POST

Response Type

Timer

Parameters

  • Body: Timer Timer(Timer)

    New timer configuration

cancelTimer

Cancels the running timer

URL

http://localhost/api/v1/timer/{id}/cancel

HTTP Method

POST

Response Type

Parameters

  • Path: id UUID

    ID of the timer to cancel

deleteAlarm

Delete an alarm from the product

URL

http://localhost/api/v1/alarm/{id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: id UUID

    ID of alarm to delete

disableAlarm

Disable an existing alarm

URL

http://localhost/api/v1/alarm/{id}/disable

HTTP Method

POST

Response Type

Parameters

  • Path: id UUID

    ID of the alarm to disable

editAlarm

Replace an already existing alarm on the product

URL

http://localhost/api/v1/alarm/{id}

HTTP Method

PUT

Response Type

Parameters

  • Path: id UUID

    ID of the alarm to update

  • Body: AlarmInstance AlarmInstance(AlarmInstance)

    New alarm instance

POST

Response Type

Parameters
