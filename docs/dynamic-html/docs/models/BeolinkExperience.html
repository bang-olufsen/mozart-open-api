BeolinkExperience

  • category : String
    Source category
    Enum:
    TV
    RADIO
    MUSIC
    MEDIA
    UNKNOWN
  • id : String
    experience id, e.g. \"RADIO\" or \"toslink\"
  • linkable : Boolean
    Linkable means \"Does pressing play on this source make sense\"
  • name : String
    Experience name, e.g. \"Bluetooth\" or \"Bang & Olufsen Radio\"
  • productFriendlyName : String
    Optional product friendly name
  • sourceFriendlyName : String
    Optional friendly name for source
  • type : String
    Source type. This corresponds to the exhaustive list or sources provided by the control layer
  • uniqueSourceId : String
    The actual device jabber id that the experience comes from
