Action

  • buttonName : String
    Name of the Button used for button-press types ('buttonShortPress')
    Enum:
    microphone
    bluetooth
    next
    previous
    playPause
    previousNext
    preset1
    preset2
    preset3
    preset4
  • deezerUserId : String
    Id of user only used for 'type=deezerFlow' and is optional
  • presetKey : String
    Only used for 'type=sourcePreset'
  • queueItem : PlayQueueItem
  • queueSettings : PlayQueueSettings
  • radioStationId : String
    Id of RadioStation only used for 'type=radio'
  • source : SourceTypeEnum
  • stopDuration : Integer
    The duration to wait (seconds) until stopping only used for 'type=stop'
  • toneName : String
    Name of the tone only used for 'type=tone'
    Enum:
    alarm_1
    alarm_2
  • type : String
    The type of Action
    Enum:
    standby
    tone
    radio
    sourcePreset
    deezerFlow
    activateSource
    playQueue
    buttonShortPress
    allStandby
    touchToJoin
    fallback
    volume
    stop
  • volumeLevel : Integer
    The volume level to configure only used for 'type=volume'
